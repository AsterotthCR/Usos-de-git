git config --global (Ejecuta la configuracion de forma global
                    para todos los proyectos)

git config --global user.name "(Agregar tu usuario)"

git config --global user.emal "(Agrega tu email)"

git config --global core.editor "code --wait" (config editor de texto)

git config --global -e (ver config de git)

git config --global core.autocrlf (true para Windows)(input mac y linux)

git config -h (todas las configuraciones)
